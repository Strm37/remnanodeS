# remnanodeS

Скрипт для установки и управления **нодой Remnawave** на сервере с удобным интерактивным меню.

Нода Remnawave — это лёгкий контейнер с Xray-core, который ставится на отдельный
сервер и подключается к уже существующей панели Remnawave. Скрипт автоматически
устанавливает Docker, скачивает образ ноды и запускает её, а затем даёт меню для
управления.

## Возможности

- Автоматическая установка Docker и Docker Compose
- Установка и запуск ноды Remnawave
- Обновление ноды до свежей версии
- Просмотр статуса и потребления ресурсов
- Просмотр логов в реальном времени
- Перезапуск / остановка / запуск ноды
- Полное удаление ноды
- Интерактивное меню — не нужно запоминать команды

## Поддерживаемые системы

- Ubuntu 20.04 / 22.04 / 24.04
- Debian 11 / 12

## Установка

Запустите на сервере одной командой (от root):

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Strm37/remnanodeS/main/remnanodeS.sh)
```

или скачайте и запустите вручную:

```bash
wget -O remnanodeS.sh https://raw.githubusercontent.com/Strm37/remnanodeS/main/remnanodeS.sh
chmod +x remnanodeS.sh
sudo bash remnanodeS.sh
```

## Перед установкой

1. Откройте панель Remnawave → **Nodes → Management → «+»**
2. Создайте ноду и укажите **Node Port**
3. Скопируйте выданный панелью **SSL_CERT**
4. Запустите скрипт — он попросит ввести порт и сертификат

> **Важно:** в фаерволе открывайте порт ноды только для IP вашей панели,
> иначе нода будет доступна всему интернету.

## Меню скрипта

| Пункт | Действие |
|-------|----------|
| 1 | Установить / переустановить ноду |
| 2 | Обновить ноду |
| 3 | Статус ноды |
| 4 | Логи ноды |
| 5 | Перезапустить ноду |
| 6 | Остановить ноду |
| 7 | Запустить ноду |
| 8 | Удалить ноду |
| 0 | Выход |
